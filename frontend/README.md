
  # Shipso Comms Frontend

  This is a Vite + React frontend generated from Figma.

  ## Prerequisites

  - Node.js 18+ recommended
  - npm (bundled with Node)

  ## Install

  From the repo root:

  - `cd frontend`
  - `npm install`

  ## Run (dev)

  - `npm run dev`

  Vite will print a local URL (usually http://localhost:5173).

  ## Build (sanity check)

  - `npm run build`

  ## Notes

  - Local dependency folders like `node_modules/` are ignored by git (see the root `.gitignore`).
